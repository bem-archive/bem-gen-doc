headline
Заголовок

Описывает (под-)заголовок раздела.

Пример использования

Для входного BEMJSON

// page.bemjson.js

{
    block: 'headline',
    content: 'Узел как аномальная джетовая активность'
}

получим HTML

<!-- page.html -->
<h1 class="headline headline_level_1">Узел как аномальная джетовая активность</h1>

Модификаторы

level

Уровень заголовка

Модификатор level задает уровень заголовка.

// page.bemjson.js
{
    block: 'headline',
    mods: { level: 3 },
    content: 'Примечание'
}

<!-- page.html -->
<h3 class="headline headline_level_3">Примечание</h3>
